## stock control 


## Descrição

A criação da API em Nest.js para controle de estoque foi motivada pela necessidade de desenvolver uma solução robusta, escalável e eficiente para gerenciar os processos relacionados ao inventário de produtos em uma empresa. 
O controle de estoque é uma atividade crítica que impacta diretamente na operação do cliente, exigindo um sistema confiável.

## Tecnologias usadas para desenvolver

- Nest.js
- Typescript
- Jest
- JWT
- Bcrypt
- Passport
- Class validator
- Swagger ( documentação )

## Conclusão

A motivação para desenvolver a API para controle de estoque está fundamentada na busca por uma solução tecnológica moderna, segura e eficiente que otimize a gestão do inventário, minimize erros, facilite a integração com outros sistemas e proporcione uma base sólida para o crescimento e inovação da empresa.
